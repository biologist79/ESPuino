# MediaHub

MediaHub is available in its own repository: **[biologist79/ESPuino-Mediahub](https://github.com/biologist79/ESPuino-Mediahub)**.

It's a lightweight, locally-run hub (Docker container) for centrally managing RFID card assignments across multiple ESPuinos — content and playback mode come from the hub instead of the local NVS configuration. See that repository for the concept doc, setup instructions, and source.

The ESPuino-side firmware support (the `MEDIAHUB` play mode, download/sync, and the "MediaHub" tab in the web UI) lives in this repository.
